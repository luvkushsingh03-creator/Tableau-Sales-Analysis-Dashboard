# Tableau-Sales-Analysis-Dashboard

# 📊 Dashboard Insights
The dashboard highlights several important insights:

# 1️⃣ Total Sales & Profit
Displays overall business performance metrics.

# 2️⃣ Sales by Region
Helps identify which regions generate the most revenue.

# 3️⃣ Sales by Category
Analyzes performance across product categories.

# 4️⃣ Monthly Sales Trend
Shows how sales change over time.

# 5️⃣ Profit Distribution
Helps identify profitable and non-profitable segments.
